# CLS Docassemble Interviews

## Overview

A repository for keeping our [Docassemble interviews](https://docassemble.clsphila.org). Typically for developing the interviews, we work out of the Playground feature in the Docassemble admin.

## Running Locally

We recommend using the [Docker Quick Start](https://docassemble.org/docs/docker.html#starting) to develop and test locally.

Once you have docker running, upload the `.yml` file and upload the templates to the templates folder.

Try "Save and Run" to see that everything is where it needs to be.

## Contributing

[Docassemble](https://github.com/jhpyle/docassemble) itself is developed with the [MIT license](https://github.com/jhpyle/docassemble/blob/master/LICENSE.txt) and contributing to that project is separate and under its [contributing guidelines](https://github.com/jhpyle/docassemble/blob/master/CONTRIBUTING.md).

We are not seeking help with developing our interviews. These are developed with our caseworkers specifically for our needs. However, we will happily accept security fixes and help with making our business logic stronger and more cohesive.
